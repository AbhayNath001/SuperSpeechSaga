## INSTALLATION
1. Clone the repository
   ```
   git clone https://github.com/TusharAMD/SuperSpeechSaga
   ```
2. Install the modules
   ```
   pip install pygame
   pip install speechrecognition
   pip install google-generativeai
   pip install pyttsx3
   ```
